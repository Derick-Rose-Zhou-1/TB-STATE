# TB-STATE model checkpoint

This repository provides the trained TB-STATE model checkpoint for manuscript
code/model availability review.

## File

- `model.pth`: trained PyTorch checkpoint for the TB-STATE model.

## Notes

This repository contains the model checkpoint only. Raw human participant-level
data and processed expression matrices are not included and should be accessed
according to the manuscript Data Availability statement.

## Citation

Please cite the associated manuscript when using this checkpoint.
